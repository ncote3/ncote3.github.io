Corona Kills Backend

A Node Backend to Covid-19 Kills

Using Node.JS, Express.JS, and Cheerio I created a web scraper that can sanitize Covid-19 data and be callable by a URL. This API allowed me to make the React front-end even faster through properly calling for the data when it is necessary. When I was more actively working on this project I intended to find or create a library that would allow for data to download in the background as users reach the site.

Population-Sim

A Python Powered Simulator

While in my senior year of high school I was able to take a class on scientific programming with Python. The Population Simulator was my end of term project culminating everything I learned in the class.
